How does Supervised Machine Learning Work?

The MakeFaire voters are generating labels used to train the Learner
  • Model: the mathematical system that makes predictions
  • Examples: items used to train the learner. Every burger you see is an example.
  • Features: signals from the examples used by the model to form predictions (in this case: the layers of the burger are the features)
  • Parameters:variables that are learned to convert example features into predictions
  • Labels: answers given by users that say whether an example is a burger or not
  • Learner: the system that adjusts the parameters by looking at differences in predictions versus actual outcome for the training examples
